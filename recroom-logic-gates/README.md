# Rec Room - Circuitos Lógicos

## 📖 Descripción
Este proyecto consiste en la realización de pruebas dentro de Rec Room y la construcción de circuitos básicos para comprender el funcionamiento de las compuertas lógicas.

## 🎯 Objetivo
Comprender el comportamiento de la lógica digital mediante simulaciones prácticas en un entorno interactivo.

## 🛠️ Actividades realizadas

- Exploración del entorno de Rec Room
- Desarrollo de pruebas dentro de la plataforma
- Implementación de compuertas lógicas básicas
- Validación del funcionamiento de los circuitos

## 🔌 Circuitos desarrollados

- AND
- OR
- NOT

## 🧪 Pruebas realizadas

Se probaron diferentes combinaciones de entrada para verificar que los circuitos respondieran correctamente según las tablas de verdad.

## 💡 Aprendizajes

- Comprensión de lógica digital
- Uso de herramientas visuales para simulación
- Importancia de las pruebas en sistemas

## 🚀 Conclusión

El uso de Rec Room permitió comprender de manera práctica el funcionamiento de las compuertas lógicas.
